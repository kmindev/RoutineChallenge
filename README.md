# RoutineChallenge
## 프로젝트 소개
일상 속 본인이 관심있는 챌린지를 웹 환경에서 유저들과 함께 참여하며, 커뮤니케이션을 통해 본인의 일상을
공유하여 좋은 습관을 유지할 수 있도록 챌린지 커뮤니티 서비스를 제공하는 웹 사이트

## 개발 기간
2023.03 ~ 2023.04 (4주)


## 기술
### Front-End
React.js

### Back-End
Spring Boot(MyBatis)

### DB
MySQL
